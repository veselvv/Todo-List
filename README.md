# Todo-List (Task Manager)


Консольный менеджер задач, написанный на **C** с использованием **PostgreSQL** для постоянного хранения данных. Проект позволяет создавать задачи, устанавливать дедлайны и сохранять прогресс между сессиями.

## ✨ Основные возможности

*   **Добавление задач**: Создание новых задач с описанием.
*   **Учёт сроков**: Возможность установки **дедлайна** (deadline) для каждой задачи (добавлено в последних коммитах).
*   **Просмотр списка**: Отображение всех активных и выполненных задач.
*   **Управление статусом**: Отметка задач как выполненных.
*   **Персистентность**: Все данные сохраняются в базу данных PostgreSQL и загружаются при повторном запуске.

## 🛠 Технологический стек

*   **Язык**: C
*   **База данных**: PostgreSQL 15 (в Docker-контейнере)
*   **Сборка**: Makefile
*   **Окружение**: Linux / WSL (сценарий `run.sh`)

## 📦 Установка и запуск

### Предварительные требования

*   Компилятор **GCC** и утилита **Make**.
*   **Docker** и **Docker Compose** (опционально) для запуска базы данных.
*   Библиотека разработки для PostgreSQL (`libpq-dev` или `postgresql-devel`).
